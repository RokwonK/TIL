# 속성/문서 구조 선택자

## 속성 선택자
CSS내에서 요소에 속성이 있다면 스타일을 줄 수 있음.  

- 
    ```css
    /*
    p태그에 class속성과 id 속성이 있는 곳에 전부 스타일을 줌
    */
    p[class][id] {
        color: silver;
    }
    ```


- 
    ```css
    /*
    p태그에 class속성의 title 값을 가지고 있는 곳에 스타일을 줌
    */
    p[class="title"] {
        color: silver;
    }

    /*
    = 대신
    ~= 뒤의 클래스명이 공백으로 구분되어 나오는지
    ^= 접두사가 같은지
    $= 접미사가 같은지
    *= 전체 클래스명 안에 포함되는지

    */
    ```


## 문서 구조 선택자

1. 자손 선택자 => 태그 태그{}
2. 자식 선택자 => 태그 > 태그{}
3. 인접 형제 선택자 => 태그 + 태그{}
    - 인접이란 것은 자신 바로 뒤에 나오는 형제를 뜻함
    - 뒷 태그에 css가 먹힘
